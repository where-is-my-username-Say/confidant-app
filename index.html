Getting a network / CORS error?

Some providers block requests sent directly from a browser. Anthropic and Gemini's OpenAI-compatible endpoint both allow it, and OpenRouter and Groq generally do too. If a preset you pick fails immediately with a network/CORS-looking error, that provider likely needs a small proxy server in front of it.
